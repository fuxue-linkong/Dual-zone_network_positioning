01 / 核心功能

一台设备，
连接整片天空

从地面坐标到轨道预测，从电波分区到摩尔斯电码 — 五大模块覆盖业余无线电日常所需。

分区定位

一键获取 GPS 坐标，实时计算 CQ Zone、ITU Zone 与 6 位 Maidenhead 网格定位，配合反向地理编码显示地址。

  • CQ Zone
  • ITU Zone
  • Maidenhead Grid
  • Geocoder

卫星过境预测

基于 predict4java 的 SGP4/SDP4 轨道计算，支持 CelesTrak / SatNOGS 三种 TLE 数据源，并行预测未来 48 小时过境。

  • SGP4
  • CelesTrak
  • SatNOGS
  • AMSAT 状态

CW 摩尔斯练习

完整 Koch 课程（26 课）+ 字符组 / 呼号 / 文本训练。AudioTrack 实时合成正弦波，可调 WPM、音调与播放模式。

  • Koch 方法
  • 呼号训练
  • AudioTrack
  • 进度跟踪

过境提醒

AlarmManager 精确闹钟 + WorkManager 每日刷新，AOS 前可配提前量提醒。重启自动恢复，支持仅白天模式。

  • AlarmManager
  • WorkManager
  • BootReceiver
  • 后台预测

天气与每日一言

高德天气 API 30 分钟自动刷新；hitokoto.cn 一言服务每日一句，失败回退本地文案池 — 让控制台也有温度。

  • 高德天气 API
  • Hitokoto
  • 本地兜底
02 / 应用预览

实时信号，
一目了然

Material 3 + Miuix KMP 设计语言，可自定义背景图与卡片透明度，让数据在指尖流动。

动态主题

从背景图自动提取色板，生成 Material 3 ColorScheme。状态栏透明跟随背景，沉浸式体验。

卡片透明度

0–100% 自由调节，背景图与卡片内容可读性平衡。scrim 遮罩反向缩放保证文字对比。

持续定位

首次定位后启动持续监听，地址解析 3 秒去抖。失败指数退避自动恢复，移动中无感更新。

卫星并行预测

SGP4/SDP4 计算分批并行，CPU 密集任务在 Default 调度器执行，结果按 AOS 排序，在境卫星优先。

01

开发端加密

明文 API Key 通过 gradle encryptSecrets 任务加密为 secrets.dat，提交入库。

02

运行时组装

应用启动时从 BuildConfig + 混淆数组 + 包名派生 三处组装主密钥，XOR 还原。

03

防重打包

ShardC 绑定 context.packageName，重打包后包名变更 → 主密钥不匹配 → GCM 认证失败。

04

CI 零配置

密文已在仓库，所有碎片已在代码中。CI 检出即构建，无需任何 GitHub Secrets。
